Sentinel Atlas

Sentinel Atlas is an open-source OSINT (Open-Source Intelligence) intelligence and mapping platform for collecting, verifying, and visualizing global events in real time.

The platform enables anonymous and authenticated users to submit geo-tagged intelligence reports which are reviewed through a multi-level moderation and escalation workflow before appearing on a public intelligence map. Sentinel Atlas is designed to support journalists, researchers, NGOs, analysts, and the public with transparent, auditable, and verifiable intelligence.
